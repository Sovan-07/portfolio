# Personal Portfolio 🌐✨

A modern and fully responsive developer portfolio website showcasing projects, technical skills, achievements, and contact information. The portfolio is designed to provide a clean and interactive user experience while highlighting development expertise, creativity, and personal branding.

This project demonstrates modern frontend development practices with smooth animations, responsive layouts, and optimized performance.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sovan-07/portfolio.git
```

## 2️⃣ Navigate to the Project Directory

```bash
cd portfolio
```

## 3️⃣ Install Dependencies

Using npm:

```bash
npm install
```

Or using yarn:

```bash
yarn install
```

---

# ▶️ Running the Project Locally

## Start Development Server

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:3000
```

---

# 🔐 Environment Variables

Create a `.env.local` file in the root directory if required.

Example:

```env
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

# 🚀 Start Production Server

npm start
